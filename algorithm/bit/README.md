### 常见位操作

* a &= (a-1): 去掉最右边的为1的位, 1100 & 1011 = 1000, 将第三位的1去掉了

* a ^ a = 0, a ^ 0 = a, 自己异或自己为0, 自己异或0为自己

* a << 1, a位左移, a=a*2

* a >> 1, a位右移, a=a/2

* a &= -a, a与运算a的补码, 得到最后一位非0的值, 5 & -5 = 1(5最右边的1是第一位,转换为十进制是1)

* a & (a-1) == 0 可以判断a是否是2的n次方, 如果a是2的n次方,那么a只有1位是1,a-1就是除了那一位后面的所有位都是1,与结果是0
